Kubernetes的DashBorad以及heapster安装教程

  • kubernetes

前言

Dashboard是一个Kubernetes的web用户接口，我们可以使用dashborad来部署容器化的应用程序到Kubernetes集群，定位容器化应用的问题所在，管理集群所占用的资源。可以通过Dashboard对集群上的应用程序有一个概览，也可以创建，修改独立的Kubernetes资源（例如部署Jobs，DaemonSets等）。举个例子，你可以扩展一个部署，开始一个滚动更新，重启一个Pod或者部署新的应用程序通过部署向导。

Dashboard也提供Kubernetes集群资源的相关信息，以及相关引起的错误。
